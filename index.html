計算ロジック

1. お釣り change = P - g を重みで配分
brother = change * T1/(T1+T2)
sister = change * T2/(T1+T2)

2. バランス基準価格
g1 = g * (1 + log(T1/T2))
g2 = g * (1 + log(T2/T1))

3. 修正した1次関数
y1 = -(g1/g2) * x + g1 ← お兄ちゃん: 需要曲線 右下がり
y2 = (g2/g1) * x + g2 ← 妹: 供給曲線 右上がり

4. 交点が均衡価格x* と均衡量y*です
